# painel-frases
Painel De Frases Motivacionais

Este projeto apresenta um painel simples e inspirador que exibe frases motivacionais para ajudar o usuário a manter o foco, a produtividade e o bem-estar ao longo do dia. Cada acesso (ou atualização) do painel mostra uma nova mensagem motivadora, oferecendo pequenas doses de incentivo de forma prática e visualmente agradável.
O objetivo do painel é servir como uma ferramenta leve e acessível para quem busca motivação rápida, seja para estudos, trabalho ou desenvolvimento pessoal.

## Funcionalidades
- Clique no Botão e gere uma frase
- Layout Responsivo

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Git e Github

## Como Rodar

        1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
		
        2. Extraia a pasta
		
        3. Abra o arquivo index.html no navegador

## Como Clonar

No terminal do VS Code digite:

	git clone https://github.com/Fellipe-07/painel-frases.git
	
No GitHub Desktop:

	No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Fellipe-07/painel-frases.git
